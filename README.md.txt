# CHINEDU AMAZING 3 MOVIE TRAILER WEBSITE

**What is the purpose of this website

The purpose of this website is to display 3 of Chinedu's favorite movie trailers.

**How to install Python 2.7.14

Open webbrowser and enter the following web address: https://www.python.org/ftp/python/2.7.14/python-2.7.14.msi

**How to download my code.

User can simply open the Movie zip file and then open the "fresh_tomatoes.py", "media.py", and "entertainment.py". User can then copy and paste all 3 codes in seperate
files. Be sure to save both files within the same folder. 

**How to open the movie trailor website

User must open the "entertainment_center.py" file with the Python Idle program. Click the run tab on the top bar of Python Idle program
and then click run module. Once the internet browser appears, the user should see 3 movie links. The user can then enjoy the experience of the website by clicking on 
the individual links to view the movie trailer.
